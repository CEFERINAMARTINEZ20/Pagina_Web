HTML 4 y 5

HTML 4 y 5 - la evolución del diseño web hasta nuestros días

HTML 4 supuso un gran salto cualitativo a la hora de diseñar página web, especialmente en lo referente a la accesibilidad web. HTML 4 ofrecía una serie de mejoras para permitir el acceso a Internet a personas con discapacidad..

Pocos años después, aquellos programadores que preguntaban cómo crear una página web más adaptada a las necesidades actuales de los usuarios, desarrollaron HTML 5, la quinta versión del lenguaje HTML y la cual se utiliza a día de hoy. Guarda un amplio abanico de ventajas respecto al diseño de páginas web con HTML 4: Integración de gráficas SVG, nuevas etiquetas para integrar de manera más sencilla contenido multimedia sin necesidad de utilizar plugins..

El reto actual de los responsables de crear páginas web está en contar con una web responsive, esto es, adaptada a los dispositivos móviles para que los usuarios puedan acceder desde ellos de una manera correcta. En rasgos generales, ésta es la breve pero intensa historia del diseño web desde sus inicios hasta nuestros días. Una cronología de hechos que si se analizan por separado pueden parecer insignificantes, pero que, observándolos en su conjunto, han permitido que el diseño de páginas web sea tal y como lo conocemos actualmente.
