HTML 3 - un nuevo paso en el diseño de páginas web

El salto cualitativo más grande entre las distintas versiones del lenguaje HTML en el mundo del diseño web se da entre HTML 2 y HTML 3. Con éste último a los encargados de crear páginas web se les abrió un mundo de posibilidades con las hojas de estilo CSS.

Así, los sitios de tercera generación ya pueden tener fondos a color e incluso introducir animaciones gracias al nacimiento del formato Gif. Es precisamente gracias al desarrollo de HTML 3 cuanto el diseño de páginas web comienza a ser una de las profesiones más valoradas..

HTML 3.0 se basa en HTML 2.0 y proporciona compatibilidad total con versiones anteriores. Las tablas han sido una de las características más solicitadas, con el flujo de texto alrededor de las figuras y las matemáticas como finalistas. Los modelos de tabla SGML tradicionales, por ejemplo, el modelo de tabla CALS, son realmente complejos. La propuesta HTML 3.0 para tablas utiliza un estilo ligero de marcado adecuado para renderizar en una amplia gama de dispositivos de salida, incluidos los sintetizadores braille y de voz.
